The Foreign, Commonwealth and Development Office (FCDO) has compiled a list of COVID-19 test suppliers abroad. British nationals can contact suppliers on the list to book a COVID-19 test before returning to the UK. You can search for Covid test providers by:

  • country
  • specific area, city or postcode
  • turnaround time of a test result

All test suppliers on the list have been contacted by the FCDO to make sure they have:

  • good English language skills
  • the same UK standards for COVID-19 testing
  • up to date contact details

We recommend you do your own research before deciding which test supplier to use.

Feedback on COVID-19 test suppliers

The FCDO welcomes your feedback if you have used COVID-19 test suppliers from the list. We use this feedback to make sure all information is up to date and accurate.

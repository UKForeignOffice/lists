The Foreign, Commonwealth and Development Office (FCDO) provides a list of Covid test providers abroad to help British nationals find a suitable Covid test provider in order to travel back to the UK. You can search for Covid test providers by:

  • country
  • specific area, city or postcode
  • turnaround time

All providers on the list have been contacted by the FCDO to make sure they have:

  • good English language skills
  • meet UK standards for testing
  • up to date contact details

We recommend you do your own research before deciding which lawyer to use.

Feedback on Covid test provider

The FCDO welcomes your feedback if you have used Covid test providers from the list. We use this feedback to make sure all information is up to date and accurate.

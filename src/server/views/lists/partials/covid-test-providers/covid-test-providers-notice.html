The Foreign, Commonwealth and Development Office (FCDO) has compiled a list of COVID-19 test providers abroad. British nationals can contact providers on the list to book a COVID-19 test before returning to the UK. You can search for COVID-19 test providers by:

  • country
  • specific area, city or postcode
  • turnaround time of a test result

All test providers on the list have confirmed that they have:

  • good English language skills
  • COVID-19 tests that meet the applicable standard (“UK standards”) for travellers returning to England, Wales, Scotland and Northern Ireland
  • up to date contact details

We recommend you do your own research before deciding which test provider to use.

Feedback on COVID-19 test providers

The FCDO welcomes your feedback if you have used COVID-19 test providers from the list. We use this feedback to make sure all information is up to date and accurate.

If you would like to send any feedback on COVID-19 test provider’s list please email digitalservicesfeedback@fco.gov.uk

The FCDO will quickly respond to feedback and will remove any supplier from the list where appropriate.
